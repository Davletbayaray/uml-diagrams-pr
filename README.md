# UML Диаграммы системы управления университетом и онлайн-банкинга

##  Описание проекта
Данный проект содержит UML-диаграммы, разработанные в рамках курса "Шаблоны проектирования приложений".

---

##  1. Диаграмма классов (Система университета)

Диаграмма классов отражает структуру системы управления университетом.

###  Основные сущности:
- Пользователь (абстрактный класс)
- Студент
- Преподаватель
- Администратор
- Университет
- Факультет
- Кафедра
- Курс
- Расписание
- Экзамен
- Оценка

###  Особенности:
- Использовано наследование: `User → Student, Teacher, Admin`
- Применена композиция: `University → Faculty → Department`
- Реализованы связи:
  - Студенты записываются на курсы
  - Преподаватели ведут курсы
  - Экзамены формируют оценки

###  Дополнительно:
- Добавлены сущности:
  - Стипендия
  - Проекты

---

##  2. Диаграмма компонентов (Онлайн-банкинг)

Диаграмма компонентов отражает архитектуру системы онлайн-банкинга.

### Основные компоненты:
- Frontend (Web и Mobile приложения)
- Backend (серверная логика)
- База данных
- Сервис авторизации
- Платежный шлюз
- Система уведомлений
- Модуль аналитики
- Система мониторинга

###  Взаимодействие:
- Frontend ↔ Backend через REST API
- Backend ↔ Database через SQL
- Backend ↔ внешние системы через API
